# Search Bangs

Ai sometimes goes wrong, why not use AI to generate search queries and filter data by ourselves?

## Example Instance

https://search.lunarclient.top/query?q=%s&engine=%s

## Supported Search Engines

| Friendly Name | Name in Query |
| ------------- | ------------- |
| Duckduckgo    | ddg           |
| Google        | google        |
| Bing          | bing          |
| Baidu         | baidu         |
